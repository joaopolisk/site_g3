Como jogar pelo terminal \o/

Siga os passos abaixo e comece a jogar!:

1. Clone o repositório no terminal
git clone https://github.com/joaopolisk/g3.git
2. Acesse a pasta do projeto
cd g3
3. Compile e execute
javac App.java
4. Execute
java App
5. Comece a jogar
Digite 1 1 ou 2 2

Como testar o projeto localmente

Siga os passos abaixo para clonar e executar o repositório:

1. Clone o repositório
git clone https://github.com/joaopolisk/g3.git
2. Acesse a pasta do projeto
cd g3
3. Compile e execute (Via Terminal)
javac Main.java && java Main
