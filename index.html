Como jogar pelo terminal \o/

Siga os passos abaixo e comece a jogar!:

1. Clone o repositório no terminal
git clone https://github.com/joaopolisk/g3.git Clique para copiar
2. Acesse a pasta do projeto
cd g3 Clique para copiar
3. Compile
javac App.java Clique para copiar
4. Execute
java App Clique para copiar
5. Comece a jogar!
Digite 1 1 ou 2 2
